# Student Performance Analytics Dashboard

A Streamlit-based analytics dashboard for recording, managing, and visualizing
student behavioral and performance observations.

## Features
- Live observation entry
- Individual student analysis
- Class-level performance analysis
- Radar and bar chart visualizations
- SQLite database integration

## Technologies Used
- Python
- Streamlit
- SQLite
- Pandas
- Matplotlib

## How to Run
```bash
pip install streamlit pandas matplotlib numpy
streamlit run student.py
